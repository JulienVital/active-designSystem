# v-ApTooltip

Directive to display tooltips on any node in the dom

## How to use

```html
<span v-ApTooltip="'my tooltip'">Span tooltip</span>
<div v-ApTooltip="'my tooltip'" 
    style="height:50px; width: 50px; backgroundColor: red;" >
</div>

```
